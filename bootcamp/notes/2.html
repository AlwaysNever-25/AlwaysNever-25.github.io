What is CSS?

Cascading Style Sheet

Used to style on the content already tagged in HTML

How to write with CSS:

                Selector { 
                    property1: value; 
                    property2: value; 
                }
  • Selector
    • Which element/class/id that CSS will affect
    • Examples for element: 
                                  body {
                                property1: value;
                            }
    • Examples for class: 
                                  .classname {
                                property1: value;
                            }
    • Examples for id: 
                                  #idname {
                                property1: value;
                            }
  • Property
    • Which property that you want to affect
    • Examples: fonts, display, background-color
  • Value
    • Values that you want for the property
    • Each property has different types of values, and so there is no general example
    • Most of the properties, however, share the same values of inherit and initial

CSS is applied in three ways:

  • Inline (style immediately inside any element tag)
    • extreme load time, usually not recommended
  • Internal ( inside head tag)
    • not recommended because it clutters the HTML file
  • External (using .css files and call it using tag inside head tag)
    • the recommended use

FONTS

Properties to style and design fonts of a website.

Important notes:

  • Fonts can be configured using a shorthand property fonts. When used that way, the font-family and font-size are required.

  • Font-family has four inputs for alternate uses. The fonts used, and alternate variants. These alternate variants can either be family-name (Arial, Helvetica) or generic-family (serif, sans-serif)

  • Font-size can alter the size of a font, from extra-extra small (xx-small) to extra-extra large (xx-large). It can also be set to a certain length (px, cm, mm, in, pt) or set according to a percentage of parent's element font size using %. It can also be generally set as smaller or larger than parents elements with smaller or larger value.

  • Font-weight can change how bold a font would be. You can use certain value like normal, bold, bolder, or lighter. You can also use a number ranging from 100 (very thin) to 900 (bolder than bold).

  • Font-variant can be used to set the font using a small-caps font. The example of this can be seen above, below the FONTS header.

  • Font-style is used to set the font using an italic style or an oblique style.
