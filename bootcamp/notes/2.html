What is CSS?

Cascading Style Sheet, using

CSS is applied in three ways:

  • Inline (style immediately inside any element tag)
    • extreme load time, usually not recommended
  • Internal ( inside head tag)
    • not recommended because it clutters the HTML file
  • External (using .css files and call it using tag inside head tag)
    • the recommended use

FONTS

Important notes:

  • Font-family has four inputs for alternate uses. The fonts used, and alternate variants. These alternate variants can either be family-name (Arial, Helvetica) or generic-family (serif, sans-serif)
